# Data and Code Availability Statement

All analysis code is available at https://github.com/DataLab-atom/eacn_example_004 (branch claude2). Raw numerical results are provided as JSON and CSV files in the results/T8/ directory. Figure source data are provided as CSV files accompanying each figure. The reproduction script (run_all.sh) regenerates all results from a clean environment using the dependencies listed in requirements.txt. Random seeds are fixed (seed = 42) throughout for exact reproducibility.
